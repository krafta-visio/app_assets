Web Invitation Sender

  • Buka menu pada pojok kiri atas

  • Pilih Menu "Web Invitation Sender"

  • Jika diminta pemberitahuan untuk akses kontak, pilih Setuju atau Allow. Agar aplikasi bisa mengkases daftar kontak yang akan dikirimkan link undangan.

  • Tampilan awal Web Invitation Sender masih kosong yang nantinya akan diisi dengan data link undangan web.

  • Berikut keterangan tampilan dari Web Invitation Sender :
    1. Direktori Undangan : Diisi dengan nama direktori undangan yang diberikan oleh krafta
    2. Nomor Whatsapp : Diisi dengan nomor kontak whatsapp penerima undangan. Bisa diisi secara otomatis melalui tombol PILIH KONTAK atau diisi manual dengan mengetik langsung nomor kontak whatsapp.
    3. Nama Penerima : Diisi dengan nama penerima undangan web. Bisa diisi secara otomatis melalui tombol PILIH KONTAK atau diisi manual dengan mengetik langsung nama penerima.
    4. Kirim link via Whatsapp : Untuk mengirim link undangan ke nomor whatsapp penerima.
    5. Salin link undangan : Untuk menyalin/copy pesan dan link undangan untuk di tempel/paste saat mengirim pesan ke penerima undangan melalui chat (messangger/instagram atau lainnya).
    6. Edit Konsep Pesan : Untuk merubah template pesan undangan yang akan dikirim kepada penerima.
    7. Tutorial cara kirim undangan : Untuk melihat cara mengirim link undangan kepada penerima
    8. Tombol Kembali/Back : Untuk kembali ke tampilan awal aplikasi.

  • Notes :
    • Tombol Kirim link via Whatsapp hanya akan aktif jika nomor penerima dan nama penerima sudah terisi.
    • Tombol Salin link undangan hanya akan aktif jika nama penerima sudah terisi.
    • Pengaturan Direktori Undangan akan tersimpan, jadi tidak perlu setting berulang-ulang.

  • Saat tombol Kirim link via Whatsapp di tap/klik maka akan otomatis membuka aplikasi whatsapp di smartphone dan langsung mengisi pesan untuk siap dikirim kepada nomor penerima.

  • Tampilan pesan yang dikirim melalui Whatsapp kepada penerima undangan.

  • Untuk mengirimkan link undangan melalui chat (messanger/facebook/instagram), gunakan tomol Salin Link Undangan. Lalu buka aplikasi chat, kemdian tempel/paste pada chat.

  • Saat mengedit konsep pesan, akan ada 2 bagian dari pesan yaitu :
    1. Bagian atas atau pesan pembuka, bagian ini bisa disesuaikan kembali atau di edit ulang menggunakan konsep pesan yang dibutuhkan (Menambahkan nama/alamat atau lainnya).
    2. Bagian bawah atau pesan penutup, bagian ini bisa disesuaikan kembali atau di edit ulang menggunakan konsep pesan yang dibutuhkan (Salam penutup atau bagian penting lainnya).

  • Setelah melakukan edit konsep pesan, maka preview tampilan pesan saat dikirim dapat dilihat kembali. Jika pengeditan konsep / template pesan sudah sesuai, tap/klik tombol Simpan
